# 📋 ЧТО ГДЕ ДОЛЖНО БЫТЬ НАПИСАНО

## 🔧 Настройка Channel Webhook / Waba Webhook:

### 1️⃣ Webhook URL (главное поле):

**Должно быть:**
```
https://untransformable-tomeka-homogonously.ngrok-free.dev/webhook
```

**Важно:**
- ✅ Должен заканчиваться на `/webhook`
- ✅ Должен быть HTTPS (начинается с `https://`)
- ✅ Должен быть из ngrok (содержит `ngrok-free.dev`)

**НЕ должно быть:**
- ❌ `trycloudflare.com` (это другой сервис)
- ❌ Без `/webhook` в конце

---

### 2️⃣ Headers (секция ниже URL):

**Должно быть добавлено:**

**Key (поле слева):**
```
X-Verify-Token
```

**Value (поле справа):**
```
terra_verify123
```

**Как добавить:**
1. Введите `X-Verify-Token` в поле "Key"
2. Введите `terra_verify123` в поле "Value"  
3. Нажмите кнопку **"+"** (синяя кнопка справа)
4. Заголовок появится в списке выше

---

### 3️⃣ Сохранение:

После заполнения нажмите кнопку **"Save"** (зеленая кнопка внизу справа)

---

## 🔍 Проверка:

После сохранения проверьте:

1. **URL правильный?**
   - ✅ Начинается с `https://`
   - ✅ Содержит `ngrok-free.dev`
   - ✅ Заканчивается на `/webhook`

2. **Headers добавлены?**
   - ✅ Видите "1 header" или "Unmask" рядом с Headers
   - ✅ Key = `X-Verify-Token`
   - ✅ Value = `terra_verify123` (может быть замаскировано как terra*****)

3. **Все сохранено?**
   - ✅ Нажали кнопку "Save"
   - ✅ Видите сообщение об успешном сохранении

---

## ⚠️ Частые ошибки:

1. **Неправильный URL:**
   - ❌ `trycloudflare.com` → должно быть `ngrok-free.dev`
   - ❌ Без `/webhook` → должно заканчиваться на `/webhook`

2. **Заголовки не добавлены:**
   - ❌ Поля Key/Value пустые → нужно заполнить и нажать "+"
   - ❌ Неправильный Key → должно быть `X-Verify-Token`

3. **Не сохранено:**
   - ❌ Заполнили, но забыли нажать "Save" → нажмите "Save"!
